# I'm attempting to make a neural network from scratch
Specifically, I'm training a model that can classify images from the CIFAR-10 dataset, using only Python with NumPy.

## My process:
My goal with this project is to really focus on the underlying mathematics, and interpretation into code, for basic neural networks. This means that I'm not using:
- AI for mathematics or implementation help
- Any coding tutorials

The biggest resource I'm using for this project is 3Blue1Brown's series on neural networks. In this series, he uses the MNIST handwritten digits dataset to explain how linear algebra, calculus, and statistics are combined to algorithmically classify images.
